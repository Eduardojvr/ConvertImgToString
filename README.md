## ℹ️ Sobre
Aplicação simples para conversão de textos presentes em imagens para formato textual, permitindo operações com o mesmo.

## ℹ️ Como usar
Navegue pelo terminal até a pasta do projeto e digite:
```Terminal
$ chmod +x dependencias.sh
$ ./dependencias.sh
$ python3 main.py
```
